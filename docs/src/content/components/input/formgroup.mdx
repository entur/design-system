# FormGroup

FormGroup er en wrapper-komponent, som legger på nødvendig label og potensiell infomelding på en input.

<Playground>
  <FormGroup label="Togdestinasjon" feedback="Gratulerer" variant="success">
    <TextField placeholder="Oslo" />
  </FormGroup>
  <FormGroup
    label="Togdestinasjon"
    feedback="Oisann, noe galt skjedde her"
    variant="error"
  >
    <TextField placeholder="Gjøvik" />
  </FormGroup>
  <FormGroup label="Togdestinasjon" feedback="Pass på her" variant="warning">
    <TextField placeholder="Advarsel" />
  </FormGroup>
  <FormGroup
    label="Togdestinasjon"
    feedback="Denne informasjonen hjelper deg"
    variant="info"
  >
    <TextField placeholder="Informasjon" />
  </FormGroup>

</Playground>
