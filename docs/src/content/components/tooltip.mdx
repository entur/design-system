# Tooltip

Installering:

```bash
npm install @entur/tooltip
# or if you are using Yarn:
yarn add @entur/tooltip
```

## Props

<Props of={Tooltip} />

## Eksempel

<Playground>
  <Tooltip content="Her er tooltipen" placement="top">
    Denne teksten har en tooltip
  </Tooltip>
</Playground>

Tooltip fungerer også i contrast theme

<Playground>
  <div className="entur-contrast">
    <Tooltip content="Her er tooltipen" placement="top">
      Denne teksten har en tooltip
    </Tooltip>
  </div>
</Playground>

Tooltip-teksten vil også bli lagt til i trigger-elementet som en aria-label.
