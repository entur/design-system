## Komponenter

### Datepicker

<ImportStatement imports="DatePicker" />
<Props of={DatePicker} />

Komponenten vår er bygget på `react-datepicker`, og støtter dermed en rekke props(som forsåvidt ligger i lista over) , selv om 100% kompabilitet ikke kan garanteres.
Les mer om disse [her](https://reactdatepicker.com/)

Den støtter også `date-fns` og det anbefales at datofunksjonene fra `date-fns` brukes.
Eksempelvis `addWeeks` for å sette en maksgrense på én uker man kan velge datoer fra.

<Playground>
  {() => {
    const [date, setDate] = React.useState();
    return (
      <DatePicker
        selectedDate={date}
        onChange={date => setDate(date)}
        minDate={new Date()} //Mindate, Fra og med
        maxDate={addWeeks(new Date(), 1)} // Til og med
      />
    );
  }}
</Playground>
