## Komponenter

### Breadcrumbs

<ImportStatement imports="Breadcrumbs" />
<Props of={Breadcrumbs} />

`<Breadcrumbs />` lager en universelt utformet brødsmulesti - bedre kjent som breadcrumbs. Den mottar en liste med lenker - hvor den første i listen er "øverst" i lenkehierarkiet (tenk forsiden), og den siste i listen er den nåværende siden.

Du kan sende inn en lenke-komponent via `as`, som vil rendre hver av lenkene. Om du bruker `react-router`, for eksempel, kan du sende inn `Link` - så ordner komponenten resten.

Listen med lenker ser slik ut:

```js
[
  { link: '/', label: 'Hjem' },
  { link: '/produkter', label: 'Produkter' },
  { link: '/produkter/togsett', label: 'Togsett' },
];
```
