## Retningslinjer for bruk

Text areas er lange tekstfelt som lar brukeren skrive tekst over flere linjer. Tekstfeltet kan justeres i høyden. Bør benyttes med label og eventuelt variant-stiler og validering. Hvis tekststyling er nødvendig, bruk Rich Text Editor for å støtte dette.
