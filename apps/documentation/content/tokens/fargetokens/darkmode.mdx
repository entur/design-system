## Hvordan ta i bruk i kode

Alle våre komponenter fra komponentbiblioteket bruker fargevariabler som støtter light, dark og contrast.

For å aktivere dark-mode på designsystemets komponenter kan du inkludere attributtet data-color-mode på html-elementet. Dette attributtet vil stå som en indikator for hvilken mode som skal aktiveres, om det er light eller dark.

```html
<html data-color-mode="dark">
<html data-color-mode="light">
```

For å bytte mellom light- og dark-modus, trenger du en måte å endre verdien til data-color-mode-attributtet dynamisk. Dette er avhengig av ditt valg av rammeverk eller bibliotek.

Om deler av din løsning ikke bruker komponentene våre må du sørge for at de har fargevariabler som støtter darkmode. Dette kan du få til ved å enten bruke variabler fra vår `base.css`, for eksempel: var(--basecolors-text-light). Eller lage et internt stilsett, eksempel:

```scss
/* intern styles.scss */

/* Stiler for light-modus */
[data-color-mode='light'],
:root {
  --my-background-color: white;
}
/* Stiler for dark-modus */
[data-color-mode='dark'] {
  --my-background-color: black;
}
```
