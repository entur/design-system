Z-index bør brukes med omhu. Overforbruk av z-indexverdier kan føre til uforutsigbar adferd og vanskeligheter med vedlikehold. For å gjøre verdiene mer lesbare er det lurt å bruke tokens med forståelige navn, som våre tokens.


```css
@use '~@entur/tokens/dist/styles.scss';

.card {
    z-index: $z-indexes-behind;
}
```
