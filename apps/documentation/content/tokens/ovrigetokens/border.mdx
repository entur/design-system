Border-variablene våre definerer tykkelse og radius på kantene til elementer. Variablene gjør det enkelt å opprettholde konsistens i løsningen din og følge Entur brandet.


```css
@use '@entur/tokens/dist/semantic.scss' as semantic;
@use '@entur/tokens/dist/styles.scss';

.card {
    border: semantic.$stroke-accent $border-widths-default solid;
}
```
