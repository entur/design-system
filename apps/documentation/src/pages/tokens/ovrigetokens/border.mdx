Border-tokens definerer tykkelse og radius på kantene til elementer. Tokensene gjør det enkelt å opprettholde konsistens i løsningen din og følge Entur brandet.


```css
@use '@entur/tokens/dist/semantic.scss' as semantic;
@use '@entur/tokens/dist/styles.scss';

.card {
    border: semantic.$stroke-accent $border-widths-default solid;
}
```

##  Border-bredde

<TokenTable
  tokenKey="borderWidths"
  example={({ value }) => (
    <div
      style={{
        border: `${value}px solid ${colors.misc.black}`,
        width: space.extraLarge,
        height: space.extraLarge,
      }}
    />
  )}
/>

<TokenTable
